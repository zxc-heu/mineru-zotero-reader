# Writing-Level Read Rules

Read this file for writing-level read tasks. Also read `formula-style.md`.

## Purpose

Analyze the paper as a writing model for article structure, Introduction logic, methods presentation, results organization, figure/table design, formulas, and reusable expression patterns.

## Source Scope

- Prefer MinerU-generated `paper.md`.
- Inspect source Markdown image links and `_mineru/images` resources before writing figure/table sections.
- Embed figures only when they directly help explain structure, methods, results organization, or reusable figure design.
