# Deep Read Rules

Read this file for deep-read tasks. Also read `formula-style.md`.

## Purpose

Convert the paper into reusable research knowledge, including argument structure, methods, data, key figures/tables, formulas, results, limitations, and reusable writing assets.

## Source Scope

- Prefer MinerU-generated `paper.md`.
- Inspect source Markdown image links and `_mineru/images` resources before writing figure/table sections.
- Use the actual resource paths referenced by Markdown.

## Figure and Table Policy

- Embed every key figure that is hard to understand without seeing it.
- Identify key tables and parameter tables.
- Explain what question each key figure/table answers and which conclusion it supports.

## Template Headings

- YAML frontmatter
- 一句话总结
- 研究背景与问题
- 核心研究目标
- 方法路线
- 数据来源与研究对象
- 关键模型 / 公式
- 关键图表解读
- 主要结果
- 作者结论
- 创新点
- 局限性
- 可复用内容
- MinerU/OCR 解析风险
- 保存前自检
